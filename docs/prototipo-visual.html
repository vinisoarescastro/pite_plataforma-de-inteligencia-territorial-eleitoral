Vínculo entre candidato cadastrado e dados eleitorais do TSE via SQ_CANDIDATO
O SQ_CANDIDATO é o número sequencial gerado internamente pelo TSE para cada candidato por eleição. Ele é a chave que liga o candidato cadastrado aqui aos votos importados do CSV do TSE. Consulte o campo SQ_CANDIDATO no arquivo CSV para encontrar o valor correto.
